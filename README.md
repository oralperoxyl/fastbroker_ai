# Telegram AI Bot

Простой локальный Telegram AI-бот на Python.

## Что умеет

- отвечает в Telegram через OpenAI API;
- хранит короткую память диалога в `data/memory.json`;
- команда `/reset` очищает память текущего чата;
- читает заметки Obsidian как базу знаний;
- помогает анализировать объявления, переписки, встречи и лиды;
- настройки лежат в `.env`.

## Запуск

```bash
source .venv/bin/activate
python main.py
```

Чтобы остановить бота, нажмите `Ctrl+C` в Terminal.

## Простые файлы для Mac

- `START_BOT.command` запускает бота двойным кликом.
- `INSTALL_AUTOSTART.command` включает автозапуск после входа в Mac.
- `STOP_AUTOSTART.command` останавливает автозапуск.

Логи автозапуска лежат в папке `logs`.

## Obsidian

Команда `/obsidian` проверяет, подключена ли папка заметок.

Бот читает `.md`-файлы из Obsidian и использует найденные заметки как контекст для ответа.

## AI-команды

- `/analyze` — анализ объявления или объекта.
- `/reply` — варианты ответа продавцу.
- `/meeting` — подвести переписку к встрече.
- `/leadplan` — план работы по лиду.
